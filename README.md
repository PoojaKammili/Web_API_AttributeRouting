Web_API_AttributeRouting

📘 Description
This is a C# ASP.NET Web API project demonstrating Attribute Routing.
The API manages Employee data and performs CRUD operations using a service layer.

🚀 Features

Create a single Employee (POST /api/Employee/Create)

Create multiple Employees (POST /api/Employee/CreateMultiple)

Get all Employees (GET /api/Employee/GetAll)

Get Employee by ID (GET /api/Employee/GetById?id=1)

Get Employees by Department (GET /api/Employee/GetByDept?department=HR)

Update Employee (PUT /api/Employee/Update?id=1)

Delete Employee (DELETE /api/Employee/Delete?id=1)

Uses a Service Layer for business logic

Clean project structure with Models, Services, and Controllers

🧱 Tech Stack

Language: C#

Framework: .NET 6 / .NET 7

API Routing: Attribute Routing

IDE: Visual Studio / VS Code
